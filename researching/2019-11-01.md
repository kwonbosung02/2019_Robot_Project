   ![201909204](../img/201909204.png)

   왼쪽 영역 : 3000 + 파란색 점 개수

   오른쪽 영역 : 4000 + 파란색 점 개수

   왼쪽 하단 영역 : 1000 + 파란색 점 개수

   오른쪽 하단 영역 : 2000 + 파란색 점 개수
   
   
   
   이와같은 값을 아두이노 시리얼 통신을 통해 count가 10씩 누적되면 보낸다
